# 🧱 LEGO Collection App

En moderne webapp for å utforske LEGO-minifigurer, se verdi og lagre favoritter.

---

## 🚀 Funksjoner

- 🔍 Søk etter LEGO figurer
- 🧱 Se informasjon og verdi
- ⭐ Lag favoritter (lagres i nettleser)
- 🖼️ Klikk for detaljvisning (modal)
- 🌙 Dark / Light mode
- 📱 Mobilvennlig design

---

## 🛠️ Teknologi

- HTML5
- CSS3 (modern UI + glassmorphism)
- JavaScript (vanilla)
- LocalStorage (lagring av favoritter)
